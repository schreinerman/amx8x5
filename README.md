# Ambiq AMx8x5 RTC Driver

This driver can be used for Ambiq, Abracom and Microcrystal ABx8x5 / AMx8x5 RTCs.

## Supported RTCs: 

### Ambiq 
- AM0805
- AM0815
- AM0805
- AM0815

### Abracom
- AB0805
- AB0815
- AB0805
- AB0815

### Microcrystal
- RV1805
